## 📌 Project Overview

CreditWise Loan Approval System is an AI-powered Machine Learning web application that predicts whether a loan should be approved or rejected based on applicant financial and demographic details.

This system simulates a real-world banking decision support tool used by financial institutions to reduce risk and automate loan approval processes.

---

## 🎯 Business Problem

Banks face challenges in:

- High loan default risk
- Manual approval delays
- Human bias in decision making
- Inefficient credit risk assessment

This ML system helps in:

✅ Fast automated loan decisions  
✅ Reduced financial risk  
✅ Data-driven approval process  
✅ Scalable banking AI solution  

---

## 🤖 Machine Learning Pipeline

The system follows an end-to-end ML workflow:

### ✔ Data Preprocessing
- Missing value handling using SimpleImputer
- Feature scaling using StandardScaler
- Categorical encoding using OneHotEncoder

### ✔ Model Used
- Logistic Regression (Binary Classification)

### ✔ Model Evaluation
- Train-test split validation
- Real-world inference pipeline
